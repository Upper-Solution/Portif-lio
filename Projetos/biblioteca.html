Administrador de Biblioteca

Projeto realizado com intuito de auto-desenvolvimento, onde um sistema de biblioteca foi criado para administrar, filtrar, adicionar e deletar livros e usuários. Este sistema permite uma gestão eficiente e organizada do acervo de uma biblioteca, facilitando tanto o trabalho dos administradores quanto o acesso dos usuários às informações dos livros disponíveis.


O sistema foi desenvolvido inteiramente na linguagem Java, aproveitando suas características robustas e seguras. Ele inclui funcionalidades essenciais, como a capacidade de buscar livros por diferentes critérios (título, autor, gênero, etc.), registrar novos usuários, gerenciar o usuários e devolução de livros, e manter um histórico detalhado das atividades.


Além disso, o projeto incorpora boas práticas de programação orientada a objetos e modularidade, garantindo que o sistema seja escalável e fácil de manter. O uso de Java também permitiu a integração com diversas bibliotecas e frameworks, potencializando as capacidades do sistema e assegurando um desempenho eficiente.


Este projeto não apenas demonstrou a aplicação prática dos conhecimentos adquiridos em desenvolvimento de software, mas também proporcionou uma experiência em resolução de problemas e aprimoramento contínuo das habilidades técnicas.
