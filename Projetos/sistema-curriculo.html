Sistema de Cadastro de Currículo

Informações do Projeto

  • Categoria: Desenvolvimento Web
  • Data do Projeto: Outubro de 2022
  • URL do Projeto: Sistema de Currículo

Tecnologias utilizadas

  • PHP
  • Javascript
  • HTML
  • CSS
  • Bootstrap
  • phpmyadmin

Projeto desenvolvido com o objetivo de facilitar a gestão de currículos de usuários. Este sistema permite que as pessoas criem um login com validação de CPF, façam o cadastro de suas informações profissionais e utilizem esses dados para gerar currículos personalizados. Todas as informações são armazenadas de forma segura em um banco de dados e vinculadas ao CPF do usuário.


O sistema foi construído utilizando uma combinação de tecnologias, incluindo PHP, JavaScript, HTML, CSS, PHPMyAdmin e Bootstrap. Isso garante uma interface amigável e responsiva, bem como uma lógica de backend robusta para gerenciamento de dados. Após o login, os usuários podem atualizar suas informações pessoais, acadêmicas e profissionais, que serão utilizadas para compor seus currículos.


Além do cadastro de currículos, o sistema também possui uma página de vagas de emprego, onde os usuários podem visualizar e se candidatar a oportunidades disponíveis. Outra funcionalidade importante é a página da conta do usuário, onde é possível visualizar e editar todas as informações cadastradas. Isso proporciona uma gestão centralizada e eficiente das informações dos usuários.


Este projeto exemplifica a aplicação prática de diversas tecnologias web e bancos de dados, promovendo não apenas o desenvolvimento de habilidades técnicas, mas também a experiência em design de sistemas de informação. A integração de diferentes linguagens e ferramentas demonstra a capacidade de criar soluções completas e funcionais para necessidades reais.
